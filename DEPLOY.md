# 🚀 DEPLOY RÁPIDO - Railway

## 5 Minutos para Colocar Seu Bot Online

### ✅ Passo 1: Push no GitHub
```bash
git add .
git commit -m "Preparado para Railway"
git push origin main
```

### ✅ Passo 2: Crie Conta Railway
- Abra: https://railway.app
- Clique "Sign Up"
- Login com GitHub

### ✅ Passo 3: Deploy
1. Clique "Create New Project"
2. "Deploy from GitHub repo"
3. Selecione `potepix/bot`
4. Railway detecta automaticamente!

### ✅ Passo 4: Variáveis
No painel Railway, clique "Variables" e adicione:

```
DISCORD_TOKEN=seu_novo_token_aqui
```

Pronto! Railway faz o resto automaticamente! ✨
